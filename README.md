# Login Form

A clean, responsive login form built with HTML, CSS, and JavaScript.

This repository contains a simple, accessible login form UI that demonstrates modern CSS layout and basic form validation using vanilla JavaScript.

## Demo

Open `index.html` in your browser to see the login form in action. No server or build step is required — just double-click the file or serve the folder with a static file server.

## Features

- Responsive layout that adapts to desktop and mobile screens
- Clean, modern UI using only HTML & CSS
- Basic client-side validation with JavaScript
- Accessible labels and focus states

## Files

- `index.html` — markup for the login form
- `style.css` — styles for layout and appearance
- `script.js` — simple validation and interactivity
- `Screenshot 2024-11-17 190246.png` — screenshot used in this README

## How to use

1. Clone or download the repository:

   git clone https://github.com/BinaryVortex/Login-Form-121.git

2. Open `index.html` in your preferred browser.

3. Enter a username and password to test the client-side validation.

## Customize

- Modify `style.css` to change colors, spacing, or fonts.
- Extend `script.js` to add real authentication calls to a backend API.
